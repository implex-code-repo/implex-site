A lightning-fast website for a reputable cybersecurity company in Ukraine

Client

Berezha Security Group is a cybersecurity services company focused on application security, penetration testing, and data security consulting.

Since 2014, the company has delivered 200+ projects for over 100 clients worldwide. Its customers come from a range of business sectors, including software engineering, fintech, retail, healthcare, media, gaming, legal and advisory, and others.

Industry: Cybersecurity
Location: Ukraine

Challenge

In 2020, Berezha Security Group rebranded as BSG. The next logical step was redesigning the brand book and the corporate website. But redesign was only part of the story. The ultimate goal was to make the website fast and SEO-friendly, keeping its Google PageSpeed Insights scores in the green 90-100 range. The upgrade was expected to be gradual, with the original URL kept intact to maintain ranking on Google.

Certain functions were to stay, too. For example, the website administrator reserved the right to change content without the programmer's or layout designer's help – just like with the previous, WordPress-powered website.

Developing a tool for pre-qualifying leads came as an extra task.

Solution

Design
Our UI/UX specialist collaborated with BSG's marketing director to create a new design fully matching the updated brand book and the client's expectations. An Implex engineer developed a sophisticated dynamic wizard form to pre-qualify leads and integrated it into a static website.
The company's blog was kept as a separate WordPress website. It was only redesigned to fit the new brand book.
Optimization
The updated website works much faster, as static pages are now generated at frontend build-time using React components.

Building a new CMS from scratch equaled budget overrun. To avoid that, we reused the repository web editor. The website admin can edit existing pages or create new ones using readily available blocks.

Results

  • A brand-new design comes with perfect usability and successfully represents the BSG brand.
  • The website scores a gorgeous 95-100 on Google PageSpeed Insights for desktop and mobile.
  • Modular code lets the website admin reuse the existing elements to create new pages.
  • An updated deployment scheme empowers the admin to make changes, deploy them in the test environment in one click, run tests, and send them to the production environment – again in just one click.
  • If an error pops up, the admin can easily roll back to any of the previous website versions.
